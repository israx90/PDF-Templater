# EDTech PDF Templater — Web Version

Conversor Web a PDF profesional con plantillas personalizables (.epd), carátulas, encabezados y estilos avanzados.

## Uso

1. Abre `PDF_templater.html` en cualquier navegador moderno
2. Configura los estilos usando el panel de configuración
3. Guarda tu plantilla como `.epd` para reutilizarla

## Características

- Plantillas EPD (importar/exportar)
- Cola de URLs con clasificación inteligente
- Carga XLS batch
- Estampado PDF
- Config avanzada: márgenes, tipografía, tablas, carátula drag-and-drop, APA, contratapa
- Imágenes de encabezado, carátula y contratapa

## Tecnologías

HTML5, CSS3, JavaScript, Font Awesome 6.4, Google Fonts (Inter, JetBrains Mono), SheetJS
